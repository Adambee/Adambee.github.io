开发过程中遇到的几个与 css 有关的小问题，参考一些资料总结一下。

1. 遮罩（lightboxes & dimmed backgrounds）

遮罩是很常见的一种布局模式，这里是一种简单的实现，鼠标移到按钮会显示遮罩效果。

.dimmed {
  top: 0;
  bottom: 0;
  left: 0;
  right: 0;
  position: fixed;
  background-color: rgba(0, 0, 0, .4);
}
.light-box {
  position: fixed;
  z-index: 1;
}

需要留意的地方，.dimmed 中 position 定位为 fixed，以保持在滚动时遮罩不会跟着走。

2. 更多的可点击区域（more clicked area）

当按钮较小时，可通过伪类元素来增加可点击区域。

.button {
  position: relative;
  &::before {
      content: '';
      position: absolute;
      top: -50%;
      left: -50%;
      bottom: 150%;
      width: 200%;
      height: 200%;
  }
}

3. 更多的光标类型（more cursor types）

在不同场景使用不同的光标类型，不要只是 cursor: pointer。

button {
  cursor: copy;
}

可尝试使用这些：no-drop | not-allowed | cell | vertial-text | move | alias | zoom-in | zoom-out | row-resize | col-resize。

4. 透明边框（transparent border）

可以用一个额外的 div 或伪类元素来实现透明边框效果，这里用元素本身的 border 来实现这种效果。

.content {
  border: 100px silid rgba(0, 0, 225, .36);
  background: rgba(142, 68, 173, 1);
  background-clip: padding-box;
}

需要注意的只有倒数第二行 background-clip，该元素设置背景是否延伸到边框下面。
border-box 背景延伸到边框外延，默认值；
padding-box 背景延伸到内边距外延，即边框下无背景，上面就是用到了这个属性；
content-box 背景裁剪到内容区外延，即内边距和边框均无背景。

5. 隐藏滚动条（hidden scorollbar）

想要出现滚动效果又不想看见滚动条，这是一种较简单的策略。

// chrome 加一个伪类
.box {
  &::webkit-scrollbar {
    display: none;
  }
}

// IE/Edge
.box {
  -ms-overflow-style: none;
}

6. 容器和头部固定，内容滚动布局

在模拟手机滚动时会用到。

.main-container {
  width: 375px;
  height: 667px;
  position: relative;
  .main {
    height: 100%;
    overflow-y: scroll;
    overflow-x: hidden;
    .title {
      height: 65px;
      position: absolute;
      top: 0;
      z-index: 1;
    }
    .text {
      margin-top: 65px;
    }
  }
}

7. 文字省略（text ellipsis）

// 单行省略
.single {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrape;
}
// 多行省略，适用于 webkit 浏览器和移动端
.multi {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 2;
  overflow: hidden;
}

8. 图片在容器中自适应宽高（auto image）

容器固定，图片自适应宽高，可用背景实现该效果，这里使用两个盒子实现。

.main {
  width: 120px;
  height: 120px;
  .image-container {
    height: 100%;
    width: 100%;
    img {
      max-width: 100%;
      max-height: 100%;
    }
  }
}
